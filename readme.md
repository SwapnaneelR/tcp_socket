# TCP Socket in TS
> This project implements a simple web socket server chat interface using CLI.

## Demonstration
> Watch this Youtube video that demonstrates all the mjor features : [YT Link](https://youtu.be/W1zcd58XUVQ)

## How to use it?
``git clone https://github.com/SwapnaneelR/tcp_socket``

Navigate to both the directories :

- install dependencies : ``npm install``
- run server and client : ``npm run dev``

## Features

- Multiple clients can connect and chat in real-time

- Simple login system (``LOGIN <username>``)

- Broadcast messaging (``MSG <text>``)

- User join/disconnect notifications

- ``WHO`` command to list connected users
